vim
===

_vimrc & color coding

# Install

Assumes this repository is cloned into `~/configuration`

 ```bash
# Create ~/.vim directory if it doesn't exist
 mkdir -p ~/.vim

# Create symlink for vimrc
ln -s ~/configuration/vim/_vimrc ~/.vimrc

# Create symlink for colors directory
ln -s ~/configuration/vim/colors ~/.vim/colors
 ```
